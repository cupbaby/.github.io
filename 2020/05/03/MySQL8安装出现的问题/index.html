MySQL8安装出现的问题

解决Navicat连接MySQL出现1251-Client does not support authentication protocol requested by server

  1. 解决办法：登录MySQL
  2. mysql -uroot -proot
  3. 使用mysql库，use mysql
  4. select user，host from user；
  5. alter user ‘root’@’localhost’ identified by ‘password’ password expire never;
  6. alter user ‘root’@’localhost’ identified with mysql_native_password by ‘root’;
  7. 刷新权限：flush privileges；
  8. 并重启服务：net stop mysql
  9. net restart msyql(这个命令我的命令是用不了，只能使用service.msc这个命令关闭和开启服务)
